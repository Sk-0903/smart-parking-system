# 🚗 Smart Parking System (AI-Based)

A modern **Smart Parking Management System** built using **Flask, SQLite, and AI (YOLO + OCR)**.

---

## 🔥 Features

* 🚗 Vehicle Entry & Exit Management
* 🤖 AI Number Plate Detection (YOLO + Tesseract OCR)
* 📷 Camera Capture Integration
* 🧾 Automatic Parking Fee Calculation
* 📊 Admin Dashboard (Live Data + Stats)
* 🎨 Modern Bootstrap UI

---

## 🛠️ Tech Stack

* Python (Flask)
* SQLite Database
* OpenCV
* YOLO (Ultralytics)
* Tesseract OCR
* Bootstrap (Frontend)

---

## 📸 Screens

* Home Page
* Register Vehicle
* Camera Detection
* Admin Dashboard

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```

Then open:
http://127.0.0.1:5000/
